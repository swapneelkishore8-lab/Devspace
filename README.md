# DevSpace: The Engineer's Workspace

DevSpace is an advanced, personalized tracking application built specifically for software engineering students. It uses a modern dark-mode glassmorphism UI to provide an aesthetic, hacker-themed dashboard for tracking your daily tasks, macro goals, academic subjects, and coding streaks.

## Features

- **Terminal Workspace**: An interactive, command-line style Quick Todo app equipped with a blinking cursor and monospaced theme, allowing you to instantly log sketchpad ideas.
- **Academic Arsenal**: Custom-built Subject Tracker that allows you to configure academic targets and visually adjust your progress through responsive metric sliders.
- **Dynamic Motivational Hero**: A rotating text array displaying industry-standard engineering mantras (e.g. "Talk is cheap. Show me the code.") over an ultra-realistic developer workspace aesthetic.
- **Professional Presence**: A fully responsive hub containing quick-access authenticated links to your professional profiles using real brand icons (GitHub, LinkedIn, LeetCode, Codeforces, CodeChef, HackerRank, GeeksforGeeks, and Gmail).
- **Persistent Storage**: All your Tasks, Goals, and Academic progress are seamlessly persisted locally using custom React hooks.
- **Interactive Calendar**: Custom-built minimalist monthly calendar grid seamlessly integrated alongside the terminal workspace.

## Tech Stack

- **Framework**: React 18 & Vite
- **Styling**: Tailwind CSS v4 featuring deeply customized glassmorphism profiles, radial gradients, and fluid typography.
- **Icons**: Lucide React & React Icons (SimpleIcons + FontAwesome).

## Setup & Local Development

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```

3. Open `http://localhost:5174/` or the designated port in your browser.
